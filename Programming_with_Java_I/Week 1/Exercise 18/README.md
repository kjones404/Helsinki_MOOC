# Exercise 18: Grades and points

Create a program that gives the course grade based on the following table.

|Points | Grade  |
|-------|--------|
|0–29 	| failed |
|30–34 	| 1      |
|35–39 	| 2      |
|40–44 	| 3      |
|45–49 	| 4      |
|50–60 	| 5      |

Example outputs:

```
Type the points [0-60]: 37

Grade: 2
```

```
Type the points [0-60]: 51

Grade: 5
```
